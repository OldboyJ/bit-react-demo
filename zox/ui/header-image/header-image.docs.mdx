## React Component for Zox website

A header section containing background image, blurb, and link to shop.

### Component usage

To use add component as shown below:
```js
<HeaderImage />
```

Modify prop values to change live:
```js live
<HeaderImage 
    text="Uplifting, wearable reminders."
    subText="made from recycled water bottles"
    linkText="Shop this week's release"
    imgSrc="https://assets.codepen.io/23274/zox-header-image.jpg"
/>
```
